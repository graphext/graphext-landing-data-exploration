# Landing Page Specifications - Graphext

## Goal

Build a new landing page for **Graphext.com** using the layout and visual styles from the homepage of [cluely.com](https://cluely.com/), adapting the content to Graphext's product (text data exploration and analysis platform).

---

## Localization

> **Note:** The landing page will initially be built in **English only**. Future localization support for additional languages (including Spanish, French, German, Portuguese, and others) is planned but **not in scope for the initial release**. The architecture should be prepared for i18n implementation later (e.g., using next-intl or similar).

### FAQ Section

#### Questions and Answers

**Q: What types of text data can I analyze?**
> A: Graphext works with any text data â€” customer support tickets, reviews, survey responses, social media posts, news articles, emails, chat transcripts, and more. If it's text, we can analyze it.

**Q: Do I need coding or data science skills?**
> A: Not at all. Graphext is designed for business users. Upload your data, and our AI handles the analysis. You focus on exploring insights and making decisions.

**Q: How does Graphext find patterns in my data?**
> A: We use advanced NLP (Natural Language Processing) and machine learning to cluster similar texts, extract topics, and detect sentiment. The algorithms learn from your specific data â€” no pre-configured rules needed.

**Q: What languages are supported?**
> A: Graphext supports 50+ languages including English, Spanish, French, German, Portuguese, Italian, Dutch, and many more. You can even analyze multilingual datasets.

**Q: Can I connect Graphext to my existing tools?**
> A: Yes! We integrate with popular CRMs, helpdesks (Zendesk, Intercom, Freshdesk), survey tools (Typeform, SurveyMonkey), and accept CSV uploads from any source.

**Q: Is my data secure?**
> A: Absolutely. We're SOC 2 compliant, use encryption at rest and in transit, and never share your data. Enterprise plans include additional security options.

**Q: How long does analysis take?**
> A: Most datasets are processed in minutes. A typical analysis of 10,000 support tickets takes about 2-3 minutes. Larger datasets may take longer but still orders of magnitude faster than manual analysis.

## 5. Responsive Breakpoints

| Breakpoint | Size |
|------------|------|
| Mobile | < 640px |
| Tablet | 640px - 1024px |
| Desktop | > 1024px |
| Large Desktop | > 1440px |

### Mobile Considerations

- Header: Hamburger menu
- Hero: Smaller title (40-48px)
- Grids: 1 column
- Cards: Full width with reduced padding
- Stats: Vertical stack
- Footer: 2 columns or vertical stack

## 6. Animations and Transitions

### Scroll Animations (suggested)

- Fade in + slide up for sections entering the viewport
- Staggered reveal for cards
- Number counting animation for statistics

### Micro-interactions

- Button hover: Scale + shadow
- Card hover: Lift effect
- Link hover: Color transition
- Accordion: Smooth expand/collapse
- Header: Blur effect on scroll

### Suggested Implementation

```css
/* Entry animation example */
@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(30px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

.animate-on-scroll {
  opacity: 0;
}

.animate-on-scroll.visible {
  animation: fadeInUp 0.6s ease forwards;
}
```
